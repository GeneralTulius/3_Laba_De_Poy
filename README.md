ЛАБОРАТОРНАЯ РАБОТА №3 ДЕ ПОЙ ИДБ-24-14
============================================================

Это программа представляет из себя оконное приложение с таймером и секундромером. По запуску открывыется окно меню, где можно выбрать прибор для использования. По выбору прибора затем открывается меню настроек окна, где надо выбрать размеры окна прибора. Если выбрать некорректные значения размера окна, то программа попросит ввести корректные значения. По вводу корректных значений открывается окно самого прибора, где можно им пользоваться.
